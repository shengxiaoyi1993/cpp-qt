# 大华NVR配置接入解码器



- 首先确保NVR的网络配置正确

![NVR_network](IMAGES\NVR_network.png)

- 登录解码器网页端，配置->网络->高级配置

  - 启用ONVIF 

  - 添加用户 user/admin123

    

  ![DECODER](IMAGES\DECODER.png)

- NVR接入设备 ，此时设备不会连接，需要重启解码器（使用账户 user/admin123）

![NVR_set](IMAGES\NVR_set.png)

![nvr](IMAGES\nvr.png)

- 解码器重启成功2分钟后，点击刷新，在NVR上显示连接成功
